# Design: 서울대 타이쿤

- **Feature**: 서울대타이쿤
- **Created**: 2026-04-01
- **Phase**: Design

---

## 1. 파일 구조

```
project/
├── index.html          # 메인 HTML (단일 페이지)
├── style.css           # 전체 스타일
├── game.js             # 게임 로직 전체
└── assets/
    └── (이미지/아이콘 필요시)
```

---

## 2. 화면 흐름

```
[타이틀 화면]
    ↓ 시작 버튼
[게임 화면]
  - 상단: 스탯바 (스트레스 / 학점 / 우정도 / 주차)
  - 중앙: 현재 이벤트 카드
  - 하단: 선택 버튼들
    ↓ 16주 완료 or 게임오버
[엔딩 화면]
  - 엔딩 텍스트 + 최종 스탯
  - 다시하기 버튼
```

---

## 3. 게임 상태 (JavaScript 객체)

```js
const state = {
  week: 1,          // 현재 주차 (1~16)
  stress: 30,       // 스트레스 (0~100, 100이면 게임오버)
  gpa: 4.0,         // 학점 (0.0~4.5, 2.0 미만 게임오버)
  friendship: 20,   // 우정도 (0~100)
  eventIndex: 0,    // 현재 주의 이벤트 순서
}
```

---

## 4. 이벤트 시스템

매주 3개 이벤트가 순서대로 등장:

```js
const weeklyEvents = [
  'bus',        // M1: 버스 줄 서기
  'class',      // M2: 수업 중 딴짓
  'lunch',      // M3: 점심 혼밥
  'teamproject',// M4: 팀플 빌런
  'noticheck',  // M5: 휴강 공지
  'festival',   // M6: 잔디광장
]
// 매주 랜덤 3개 선택
```

---

## 6. 구현 순서

1. HTML 뼈대 + CSS 기본 스타일 (타이틀/게임/엔딩 화면)
2. 스탯 표시 UI (스트레스바, 학점, 주차)
3. 이벤트 카드 렌더링 + 선택 버튼
4. 6개 이벤트 데이터 + 선택지 결과 로직
5. 게임오버 / 엔딩 판정
6. 전체 게임 루프 연결
7. CSS polish (픽셀아트 or 귀여운 카드 스타일)
